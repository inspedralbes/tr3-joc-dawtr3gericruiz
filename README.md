# Projecte: Videojoc 2D Multijugador amb Unity i Node.js

## Nom dels integrants
* Eric Ruiz

## Nom del projecte
* Jujutsu Bros Online

## Petita descripció
Aquest projecte consisteix en la creació d’un videojoc 2D multijugador tipus "Smash" amb Unity, connectat a un backend extern desenvolupat en Node.js. El servidor gestiona la comunicació entre jugadors mitjançant una API HTTP (Express) i WebSockets natius per a l'acció en temps real de la partida. A més, inclou persistència de dades aplicant el patró Repository (registre d'usuaris, partides i resultats) i la integració d'un agent d'intel·ligència artificial (ML-Agents) dins el joc com a rival.
